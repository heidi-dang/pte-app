# Content Policy

## Permitted Content

- Original human-written content
- Original AI-assisted content that has been human-reviewed
- Original company-created datasets
- Public-domain content
- Creative Commons Zero (CC0) content
- Creative Commons Attribution (CC BY) content with proper attribution
- Properly licensed commercial assets
- Commissioned audio recordings and images
- Synthetic audio with commercial permission

## Prohibited Content

- Copied competitor question banks
- Pearson paid preparation content
- Recalled live PTE examination questions
- Unlicensed news articles
- Unlicensed textbook passages
- Random image-search results
- Copied YouTube lesson material
- Unverified social-media question collections

## Required Publication Information

Every published content item must include:

| Field | Description |
|-------|-------------|
| Author or generator | Who created the content |
| Source | Where the content originated |
| Licence | Licence type under which it is published |
| Commercial-use status | Whether commercial use is permitted |
| Reviewer | Person who reviewed the content |
| Review score | Numerical review score out of 10 |
| Version | Version number (incremented on revision) |
| Correct answer | Official correct answer(s) |
| Explanation | Explanation of the correct answer |
| Difficulty | Difficulty level (Easy, Medium, Hard) |
| Skills | PTE skills assessed |
| Publication status | Draft, In review, Published, Retired |

## Review Gate

- No content below 9/10 may be published.
- Content scoring 9/10 or above may proceed to publication.
- Content rejected during review must include reason and revision guidance.
- The overall development phase still requires 98/100 audit score.
