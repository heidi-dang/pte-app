# Product Requirements

## Product Vision

Build a complete commercial online PTE Academic learning platform that improves real student performance through:

Diagnose → Teach → Practise → Explain → Review → Repeat → Test → Calibrate

## Core Product Modules

- Public website
- Student portal
- Teacher portal
- Content portal
- Administration portal
- Course and lesson engine
- Question-practice engine
- Speaking recorder
- Writing editor
- Objective scoring
- AI-assisted speaking and writing analysis
- Diagnostic assessment
- Personalised study plans
- Section tests
- Full mock exams
- Mistake notebook
- Vocabulary notebook
- Spaced repetition
- Subscription billing
- Teacher review
- Content production
- Content licensing
- Calibration and educational analytics

## Launch Product

Scope for the first commercial release:

- PTE Academic
- English interface
- Responsive web application
- Progressive Web App capability
- All current PTE Academic task types
- Original and licensed questions
- AI-estimated scoring
- Optional teacher review
- Paid subscriptions and practice packs

### Out-of-Scope for the First Commercial Release

- Native iOS application
- Native Android application
- PTE Core
- IELTS
- TOEFL
- Public student discussion forum
- Live video classroom
- White-label school system

These can be considered after the stable commercial launch.

## Success Metrics

### Primary Metric

Percentage of students who reach or exceed their required official PTE score.

### Supporting Metrics

- Diagnostic-to-latest score improvement
- Official-score improvement
- Estimated-to-official score difference
- Student retention
- Study-plan completion rate
- Mock completion rate
- Score improvement by task type
- Teacher-to-AI disagreement rate
- Time required to reach target score
