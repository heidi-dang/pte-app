# Release Criteria

## Final Release Requirements

The platform is ready for commercial release only when all of the following criteria are met:

1. **All task types implemented**: Every PTE Academic task type documented in the task blueprints is implemented and tested.

2. **Minimum content bank approved**: A minimum of 500 approved and published questions across all task types.

3. **Five complete mocks**: At least five full-length mock exams are available and tested.

4. **Mobile speaking recorder working**: Speaking recording functions correctly on mobile Safari and Chrome browsers.

5. **Writing autosave working**: Writing responses autosave within 5 seconds and survive page refresh, API restart and browser restart.

6. **Mock recovery working**: Interrupted mock exams can be resumed from the exact point of interruption.

7. **Payment lifecycle working**: Subscription purchase, cancellation, renewal and refund flows are fully functional and tested.

8. **Teacher portal working**: Teachers can view students, review responses, provide feedback and enter scores.

9. **Content provenance complete**: Every published content item has documented author, source, licence and review history.

10. **Scoring calibration documented**: Calibration process has been executed and results are documented.

11. **Backup restoration tested**: Database and file storage backup and restoration procedures have been verified.

12. **Monitoring working**: System health, error tracking and performance monitoring are operational and alerting correctly.

13. **Required tests passing**: All unit, integration, end-to-end and stability tests pass.

14. **Pilot completed**: At least one pilot group has completed the full platform experience and feedback has been addressed.

15. **Final audit at least 98/100**: The final release audit scores 98/100 or higher.
